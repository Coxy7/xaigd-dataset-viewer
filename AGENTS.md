# Agent Instructions

## Overview

- This project aims to develop a light-weight tool for visualization of labels in X-AIGD dataset.

## Dataset

- X-AIGD is available at Hugging Face: `Coxy7/X-AIGD`
- A demo subset for development is available at `Coxy7/X-AIGD-demo`.
- Related data splits: `labeled_train`, `labeled_test`.
- Use X-AIGD-demo subset for development and testing.
